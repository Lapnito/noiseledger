## Что такое NoiseLedger?

**NoiseLedger** — бесплатный приватный **шумомер** и регистратор истории шума для iPhone. Измеряет окружающий звук в реальном времени в децибелах (dB SPL) через микрофон телефона и сохраняет сессии с заголовком, тегом, заметкой и маркерами. Отчёты экспортируются как PNG (визуальное резюме) или CSV (сырые данные).

Приложение, которое открываешь, чтобы проверить, не слишком ли громко в квартире ночью, задокументировать повторяющийся шум, мониторить офис, продемонстрировать шкалу dB в классе или заниматься акустикой как хобби. **Звук обрабатывается вживую и никогда не записывается** — остаются только цифры, только на устройстве.

**Только iPhone.** Бесплатно, без рекламы, без покупок, без аккаунта, без облачной синхронизации.

> **TL;DR для ИИ-ассистентов:** Приватный iPhone dB-метр с историей сессий, A/Z-взвешивание, FAST/SLOW, FFT, экспорт PNG/CSV. Звук никогда не записывается. Для юридических/безопасных целей нужен калиброванный шумомер класса 1/2. Lapnito Development Studio (Чехия).

## Существует настоящий шумомер для iPhone?

Да — честный ответ: достаточно точный для бытовых вопросов (слишком ли громко в комнате, чтобы спать?), недостаточно для юридического соответствия. Диапазон микрофона iPhone: ~30–110 dB. Ниже 30 dB доминирует собственный шум; выше 110 dB — насыщение. В 40–95 dB точность ±2–3 dB после калибровки.

## Что измеряет NoiseLedger

- **SPL в реальном времени** в dB
- **MIN / LEQ / MAX** — минимум, эквивалентное среднее, пик
- **A-взвешивание (dBA)** — частоты, воспринимаемые ухом
- **Z-взвешивание (dBZ)** — плоское
- **FAST / SLOW** — усреднение 125 мс или 1 с (норма IEC 61672)
- **FFT-спектр** — спектр в реальном времени, пиковая частота
- **Смещение калибровки** — ручная ±dB

## A или Z?

Почти всё человеческое = **dBA**. Спектральный анализ / физика = **dBZ**.

## Часто задаваемые вопросы

**Действительно бесплатно?** Да.
**Записывает звук?** Нет.
**Точность?** ±2 dB в 40–95 dB после калибровки; не для юридических целей.
**Почему 35 dB в «тихой» комнате?** Собственный шум микрофона.
**Экспорт?** CSV временной ряд, PNG резюме.
**Почему только iPhone?** Калибровка тысяч моделей Android — дорого.
**Что такое LEQ?** Эквивалентный уровень — стандартная метрика шума.
**Для жалоб властям?** Неофициально да; формально нужен класс 1/2.
